# Kirnagram Dashboard

Admin dashboard for Kirnagram built with React, Vite, TypeScript, shadcn-ui, and Tailwind CSS.

## Setup

```sh
cd kirnagram-dashbord
cp .env.example .env
npm install
npm run dev
```

## Admin Credentials

Hardcoded dashboard credentials have been removed from source.

Set these values in `.env`:

- `VITE_MAIN_ADMIN_EMAIL`
- `VITE_MAIN_ADMIN_PASSWORD`
- `VITE_CLIENT_ADMIN_EMAIL`
- `VITE_CLIENT_ADMIN_PASSWORD`

## Scripts

- `npm run dev`: Start local development server
- `npm run build`: Production build
- `npm run preview`: Preview production build
- `npm run lint`: Run eslint
- `npm run test`: Run tests once
- `npm run test:watch`: Run tests in watch mode

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn-ui
